# Solana Staking Deep Dive

Welcome to the comprehensive guide on Solana staking mechanisms. This documentation covers three key components of the Solana staking ecosystem:

## What You'll Learn

### 1. Stake Program
The native Solana staking program that handles delegation of SOL to validators. Understanding this is fundamental to all other staking mechanisms.

### 2. SPL Stake Pool Program
Liquid staking solution that allows users to stake SOL and receive liquid staking tokens (LSTs) in return. This enables DeFi composability while earning staking rewards.

### 3. Jito StakeNet Program
Advanced validator management system that includes the Steward Program for automated stake delegation based on validator performance metrics.
